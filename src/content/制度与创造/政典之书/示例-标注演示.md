## 消歧功能

对于同名或简称的实体，可以使用消歧语法：

- 〖person:始皇|秦始皇〗：显示"始皇"，但链接到"秦始皇"的完整信息
- 〖person:武王|周武王〗：显示"武王"，实际指向"周武王"

## 使用说明

### 标注语法

所有实体标注使用统一格式：`〖标签:内容〗`（旧符号语法仍兼容）

- 类型符号：@人名、=地名、;官职、%时间、&朝代等
- 消歧格式：`〖标签:简称|全名〗`

### 支持的实体类型

| 符号 | 类型 | 示例 |
|------|------|------|
| @ | 人名 | 〖@孔子〗 |
| = | 地名 | 〖=长安〗 |
| ; | 官职 | 〖official:丞相〗 |
| % | 时间 | 〖time:元年〗 |
| & | 朝代 | 〖&汉朝〗 |
| ' | 邦国 | 〖state:齐国〗 |
| ^ | 制度 | 〖^科举〗 |
| ~ | 族群 | 〖tribe:匈奴〗 |
| # | 身份 | 〖identity:士人〗 |
| • | 器物 | 〖•宝鼎〗 |
| ! | 天文 | 〖!北斗〗 |
| ? | 神话 | 〖myth:盘古〗 |
| + | 生物 | 〖biology:麒麟〗 |
| $ | 数量 | 〖quantity:百万〗 |
| { | 典籍 | 〖{论语〗 |
| : | 礼仪 | 〖ritual:冠礼〗 |
| [ | 刑法 | 〖legal:五刑〗 |
| _ | 思想 | 〖_仁义〗 |

### 段落编号

在段落开头或任意位置使用 `[数字]` 或 `[数字.数字]` 即可创建段落编号：
- 单级：[1], [2], [3]...
- 多级：[1.1], [1.2], [2.1]...

### 最佳实践

1. **保持一致性**：在同一篇文章中，对同一实体使用相同的标注方式
2. **使用消歧**：当实体有简称或别名时，使用消歧语法确保准确性
3. **适度标注**：不必对每个词都标注，重点标注关键实体即可
4. **段落编号**：重要段落建议添加编号，便于引用和讨论

## 技术说明

本系统基于Astro框架的Remark插件实现，完全兼容原有的Markdown语法。新增篇目时，只需按照上述语法编写.md文档，系统会自动处理实体标注和段落编号。

样式支持亮色/暗色主题自动切换，响应式设计适配各种设备屏幕。
