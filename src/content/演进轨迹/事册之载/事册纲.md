# 事册纲

## 大事纪要
> 人类文明诸大事纪要于本章。何者为大，非帝王权霸、非名媛佳妓，非惟名实相应、术道相成、心物相通者不足为大也，显当以下标准而论：
+ 甲、革乱鼎秩，永绝旧弊：此革除迷乱，鼎立秩序，且深入人心，使除旧弊之志恒传后世，而令旧弊永世无以复辟独大；
+ 乙、普惠众生，弗求自得：普遍实惠予有情众生，而非出于牟已之所得也；
+ 丙、拯厄扶危，解民倒悬：于患难中自觉承命，躬行救苦，以解万民之倒悬；
+ 丁、开域拓知，为不可为：开辟领域，拓展认知，为古之不可为者，以成文明发展之新路也；
+ 戊、启迪心识，灵光普照：启迪本心神识，性灵显现之光普照于世；
+ 己、薪火永续，文明继绝：总成万有，抑邪扬正，以澄明文明大道之本义，令不绝于世也；
> 如此标准，纪要大事者凡一项：
- 一、提身挻立：远古先民始能直身而立，解放双手以触万物、制工具，视野遂开，心智始扬。由匍匐而挺立，非惟形貌之变，实为精神之觉醒，标志着人类主动抗争自然、开创文明之伟志初立。此合甲项“革乱鼎秩”（革除禽兽俯行之旧态）、戊项“启迪心识”之深义。
- 二、名有万物：命物定名，使混沌得序，万象各有归称。语言由是生，思维藉以精，集体之智可传，经验之粹得积。无名之天地化为有名之世界，人类得以共织意义之网。此合丁项“开域拓知”、戊项“灵光普照”之根本。
- 三、作巢生火：构木为巢以避禽兽，钻燧取火以御寒夜。由是居有所安，食得以熟，疾疫渐少，夜幕亦明。生存之基由是固，群落之聚由是稳，文明得获最初之盾与光。此合丙项“解民倒悬”、乙项“普惠众生”之实绩。
- 四、比类取象：观天地鸟兽之纹，察万物变化之机，以比拟联想而通其理。此抽象思维之初萌，符号创造之始端，为后来文字、易理、艺术诸般文明瑰宝奠定思辨之基。此合戊项“启迪心识”、丁项“为不可为”之先声。
- 五、观天制历：仰观穹宇，察日月星辰之运行；俯察物候，悟寒暑春秋之更迭。终制历法以授农时，定年月以序社会。农耕由是兴，祭祀由是明，时空之律初为人识，宇宙之秩序始为人用。此合甲项“鼎立秩序”、乙项“普惠众生”之宏业。
- 六、一画开天：伏羲画卦，肇启文明。最简单之爻画，包蕴最丰富之象数，示阴阳消长之变，通天人感应之理。此符号系统之极致，为哲学、数学、玄学之共同源头，开辟文明之道路。此合丁项“开辟领域”、戊项“灵光普照”之玄奥。
- 七、医药农桑：神农尝百草以疗民疾，教耕稼以续民命。由是疾病有方医，饥馑有术防，族群繁衍得赖保障，文明延续始有根基。生生之德，厚载万物，此合丙项“拯厄扶危”、己项“薪火永续”之仁术。
- 八、易货于市：首创以物易物之市集，使各得其所余，各补其所缺。劳动价值始得以通，社会分工由是而进，经济生活之萌芽破土而出。此合乙项“普惠众生”、甲项“革乱鼎秩”（变革原始平均之乱）之智慧。
- 九、律法初成：始制有形之规以约众行，息争止纷，由习惯而成条文，由 tribal 而及城邦，初立正义之基，使强不凌弱，众不暴寡，虽未尽善，然文明由是始入秩序之轨，人心知所向避，社会得以稳立。此合甲项“革乱鼎秩”、丙项“拯厄扶危”之旨。
- 十、轮舆致远：制轮作舆，初破空间之阻，货物得以流通，远地得以相接，民始知九州之外犹有九州。行路之难顿减，交流之域骤扩，文明非复孤岛，而渐成网络。此合丁项“开域拓知”、乙项“普惠众生”之实。
- 十一、书同文轨：离口耳相传之限，以符号固言语、载思想，纵千载之下、万里之外，前人智慧、异域见闻皆可循迹而得。文脉由是不断，经验得以累积，文明进化由线性而加速为指数。此合戊项“启迪心识”、己项“薪火永续”之宏效。
- 十二、勾股演数：发宇宙数量之秘，立推理求证之法，不以实用为限，而究纯粹之理。由是人事之外别有自然之律，人心之中渐育理性之神，文明因得脱离蒙昧直觉，进至抽象思辨之境。此合丁项“为不可为”、戊项“灵光普照”之深意。
- 十三、扬帆沧海：舍安就险，以人力抗风波，探索未知之域。非惟觅新土、通有无，更拓人心之视野，破地域之局限，知天地之广、物种之繁，文明由是真正走向整体。此合丙项“解民倒悬”（开拓生存空间）、丁项“开辟领域”之勇毅。
- 十四、仁礼立极：立仁爱为本、礼序为用之教化体系，使族群得脱野蛮之争，构建伦理共识。人心始知敬天爱人、尊长恤幼，社会由散漫而入协同，文明遂有自觉之道德根基。此合甲项“革乱鼎秩”、戊项“启迪心识”之宏规。
- 十五、机杼织造：创纺织之技，解民寒肤之困，更启工艺革命之端。衣冠既备，非惟遮体御寒，亦使礼制有依、文明有表。桑麻棉丝皆化为用，民生得以厚养，技艺由此精进。此合乙项“普惠众生”、丁项“开域拓知”之实绩。
- 十六、百工典章：汇工匠技艺为系统典章，使经验得传、标准可循。金石土木皆成法度，营造冶铸皆有规程，文明由是突破师徒口耳之限，技术得以累积迭代。此合己项“薪火永续”、戊项“灵光普照”之深谋。
- 十七、大宪约权：以成文之法约束权柄，立“王在法下”之原则，破“朕即国家”之迷思。权力始受公义之羁縻，民权由此萌蘖，政制由独断而渐趋公议。此合甲项“永绝旧弊”、丙项“解民倒悬”之壮举。
- 十八、蒸汽轰鸣：释天地蕴藏之力以代人力，机器轰鸣而万象一新。交通生产为之剧变，时空尺度为之重构，文明自农耕而跃入工业之纪元。此合丁项“为不可为”、乙项“普惠众生”之雷霆。
- 十九、电磁化通：窥破电磁奥秘，化无形之力为光明动力、迅捷通信。万里之遥缩于咫尺，黑夜如昼尽显人智之辉煌。此合戊项“灵光普照”、丁项“开辟领域”之奇功。
- 二十、万物互联：织数字之网覆裹全球，信息如气血贯注文明全身。知识壁垒为之崩塌，协作效率臻于极致，人类首次结为实时共振之有机整体。此合己项“文明继绝”、丙项“拯厄扶危”（如疫情中的全球信息共享）之新章。
- 二十一、星海启航：挣脱地心引力之缚，置足寰宇之外。月球尘壤初印人类足迹，深空探测器携文明讯号飞向星海。地球文明由此升维为星际物种，既知宇宙之浩渺，更惜家园之珍贵。此合丁项“开域拓知”、戊项“启迪心识”之远航。
