# 乙部：演进轨迹

## 时序脉络

乙部为文明之演进轨迹，以时空为纬，以因果为经，示文明发展之脉络。

### 部序

文明之发展，如长河之奔流，有源流，有治乱。乙部录文明演进之迹，盖所以明文明之变迁，知文明之规律也。

### 分部说明

乙部下设五门：

1. **肆曰 时序纲**：建纪元，划世代。系万事于岁，标转折于历，示长河之干、分期之节。

2. **伍曰 事鉴记**：以事为纬，以因为经。详载文明大变、转折之决，重续贯之析，特立“关键之择”“转折之考”。

3. **陆曰 民生志**：承《食货志》之统，载录民生之易，财货之变，社会之流。

4. **柒曰 交流鉴**：录内外之交、对话之迹、争融之实。存使节之记、商途之志、战伐之纪、文化相滋之录、言语相触之史，为星际往鉴。

5. **捌曰 技术谱**：纪器用之进、技艺之迁。自石兵至星槎，分科述理，载突破之要、代际之跃，立“数革”“智纪”“星技”“生术”诸卷。

### 编撰原则

乙部之编撰，以"通变"为要。凡所录者，必明其因果，必见其脉络。通古今之变，成一家之言，以期为文明之发展提供借鉴。

### 预期价值

通过对文明演进轨迹的研究和记载，我们可以更好地理解文明发展的规律，预测文明的未来走向，为人类的决策提供参考。
