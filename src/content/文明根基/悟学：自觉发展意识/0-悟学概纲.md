# 悟学概纲

## 一、悟学之定义

**悟学**，乃超越哲学范畴、比哲学更精深全面之新学问层次也。

何以名之曰"悟"？盖"悟"者，觉悟、悟道、顿悟之谓也。《说文解字》云："悟，觉也。"悟者，通也、明也、入也。"悟"字兼含中国儒释道三学之精髓：儒曰"悟吾心之全体大用"，道曰"悟道法自然之玄妙"，释曰"悟诸法实相之真空"。是故悟学者，以"悟"为方法、以"真"为究竟之学问也。

**悟学核心命题**：通过身心体悟，对所有学问求极限，不断逼近真理。

## 二、悟学之定位

### （一）悟学与哲学之关系
悟学统摄一切学问，哲学乃悟学在一定前提（理性主义框架）下之特例也。

- 哲学者，爱智之学也，以求知为宗、以主客二元为框架；
- 悟学者，以身心体悟对所有学问求极限、不断逼近真理之学也；
- 哲学问"何以为真"，悟学问"何以为本"；
- 哲学求"知之明"，悟学求"悟之彻"。

### （二）悟学非哲学之一分支，乃哲学之一般化
- 哲学以理性为方法、以主客二元为框架、以认识论为核心；
- 悟学以身心体悟为方法、以天人合一为境界、以本真论为核心。

### （三）悟学统摄一切学问
不独哲学，科学、宗教、艺术、伦理、政治、经济——凡一切学问，皆可从悟学之视角观之、皆可于悟学之境中证之。悟学如大海，纳百川之水；悟学如虚空，容万物之有。

### （四）悟学之范围
悟学之范围，广矣大矣：
1. **悟本论**：探一切存在之本源
2. **悟体论**：究宇宙人生之全体
3. **悟用论**：明万法运作之机枢
4. **悟境论**：臻境界修养之极致
5. **悟行论**：示修悟实践之功夫
6. **悟真论**：达真理极限之彼岸

合此六论，方成悟学之全体大用。

## 三、悟学之方法论

悟学之方法，非止于理性推演，更重身心体悟；有四要焉：

### （一）身心体悟
非徒用头脑理解，更以身心实证。《道德经》云："为学日益，为道日损。"体悟者，损去我执、消融主客、与道合一之功夫也。

### （二）极限思维
对任何学问，不止于其边界，而超而入之，求其极限。《老子》云："玄之又玄，众妙之门。"极限处，真理显焉。

### （三）统摄会通
不偏执一端，不割裂全体。以整体之眼观照万有，以通达之心融会诸学。《易》云："一致而百虑，殊途而同归。"

### （四）逼近真理
真理非一蹴可几，乃渐次逼近。《大学》云："止于至善。"至善者，极限之境也；止之者，不断趋近之谓也。

## 四、悟学宣言

今当以悟学之名义，向天下宣告：

**一、悟学之根本主张**：人之所以为人，非由形体所命，乃由自觉所成也。整体自觉意识，乃人之所以为人的本体根基；善方成人，乃人之所以为人的存在判准。

**二、悟学之最高理想**：人与人和谐共生、人与自然和谐共生、人与宇宙和谐共生——三界并立、万物一体，此悟学之最高理想也。

**三、悟学之实践路径**：由个体而至群体、由个人而至人类、由当下而至永恒——循此路径，人类可以从"自然人"进于"正人"、从"正人"进于"真人"焉。

**四、悟学之时代使命**：当今之世，人类面临前所未有之挑战——环境崩坏、价值混乱、意义危机。唯有悟学之智慧，方能引导人类走出困境、走向光明焉。

天下之人，无论肤色之异、国别之殊、文化之别，皆同属人类之一员，皆有自觉发展之可能。吾人当以此悟学之智慧，与天下之人共勉之、并行之，以成就真正之"人"焉。

是故，悟学非一家之私学，乃天下之公学也；悟学非一时代之学，乃万世之学也；悟学非一民族之学，乃人类之学也。

汝其勖哉！吾人共勉之！

## 五、悟学视角下之人观

悟学之要，在于以整体自觉之眼观照万有，以统摄会通之智穷究真理。以此方法论审视"人"之问题，则见诸家之偏、得本论之正。

哲学以主客二元框架论人，或执形体、或执心性、或执道德，皆只见一隅也。悟学以身心体悟论人，则知人之整体本质不在形体、不在心性、不在道德，而在**整体自觉意识**也。此意识使人超越禽兽、知万物一体、达天人合一之境。

## 六、悟学之人间立场

悟学立乎人间，以"人"为中心，以"悟"为方法，以"真"为究竟。其旨在于：

1. **破偏执**：破除唯物、唯心、唯道德、唯理性等诸家之偏，主张整体观照；
2. **立本真**：确立人之整体本质，明辨人之所以为人之所在；
3. **开新境**：整合诸学之精华，超越诸学之局限，开创学问之新境界；
4. **践大道**：以身心体悟之功夫，求极限真理之究竟，化民成俗，臻于大同。

悟学者，非空谈之学，乃实践之学也；非独善之学，乃觉他之学也；非一时之学，乃永恒之学也。愿天下学人，共悟此道，共践此学，共成此德，共臻此境。

## 七、悟用论示例：世间争斗之根源

悟用论者，明万法运作之机枢也。以下以"世间争斗"之解析，示悟学统摄会通之智。

**原命题**：世间种种争斗，是人的思想在斗，是人的基因在斗，是基因的意识在斗，是引力的角逐，是阴阳的转化。

此命题揭示争斗之五层根源，悟学各论可为之作系统解释：

| 争斗层面 | 悟学理论支撑 |
| --- | --- |
| 人的思想在斗 | **人正论**论认知相对性：人们各执己见、相互争战，形成治乱循环；**真理本义论**示众说纷纭之根——皆盲人摸象，各执一隅 |
| 人的基因在斗 | **广义生命论**论生命层级递进：一切存在皆为生命，皆有生存本能；基因维护自身存在、复制传递，乃争斗之生物基础 |
| 基因的意识在斗 | **自觉意识论**论一切存在皆有自觉意识：基因之"意识"即其维护自身之内在驱力；自觉意识作为内部类力，自生性、指向性、持续性 |
| 引力的角逐 | **广义生命论**论宇宙层级能量利用：星系以引力势能运作，宇宙层构中一切"角逐"皆根植于物质层面的基本作用力 |
| 阴阳的转化 | **发展存在论**论内在矛盾为发展原动力：有限与无限、秩序与混乱、个体与整体之矛盾推动一切发展，对立面转化即阴阳之道 |

**悟学统摄之洞见**：一切争斗的本质，是**不同层级自觉意识之间的矛盾运动**。从基因的意识力，到人的思想力，再到宇宙的引力，最终统一于宇宙层构分形演化中自觉意识不断扩展的天命进程。悟学者，知此根本，故能超越争斗、归于和谐焉。

### 因果链律：意识决定现实

**原命题**：意识类型决定思维模式，思维模式决定思想结构，思想结构决定行为内容，行为内容决定现实格局。

此命题揭示意识至现实之因果链条，悟学各论可为之作系统论证：

| 因果环节 | 悟学理论支撑 |
| --- | --- |
| 意识类型 → 思维模式 | **自觉意识论**论三重境界：知觉、觉察、觉悟——意识类型不同，认知广度深度遂异，思维模式因之而分 |
| 思维模式 → 思想结构 | **人正论**论认知相对性：思维模式决定认知范围与标准，进而塑造思想框架；**真理本义论**示真理唯一性如何约束思想之边界 |
| 思想结构 → 行为内容 | **发展存在论**论内在矛盾驱动选择：思想结构决定何为"利"、何为"害"，行为因之而生；**人正论**揭示异化状态下行为失范之根由 |
| 行为内容 → 现实格局 | **广义生命论**论层级涌现：个体行为汇聚为集体行动，涌现出单体所无之新性质；**宇宙天命论**示秩序崩坏与重建之道 |

**悟学统摄之洞见**：此因果链律揭示**存在之迭代规律**——意识为因，现实为果；现实又生新意识，新意识又造新现实，循环往复，层层递进。自觉意识之扩张，即此因果链不断升级之动力；悟学之目标，即令此链条趋近真理、归于和谐焉。
